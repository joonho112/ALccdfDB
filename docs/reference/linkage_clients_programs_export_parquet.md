# Export linked clients-programs data to Parquet

Export linked clients-programs data to Parquet

## Usage

``` r
linkage_clients_programs_export_parquet(obj, path, verbose = TRUE)
```

## Arguments

- obj:

  An `alccdf_linked_clients_programs` object

- path:

  Output file path

- verbose:

  Logical; print progress messages? Default TRUE.

## Value

Invisible file path
