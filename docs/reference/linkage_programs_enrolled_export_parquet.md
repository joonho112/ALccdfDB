# Export linked programs-enrolled data to Parquet

Export linked programs-enrolled data to Parquet

## Usage

``` r
linkage_programs_enrolled_export_parquet(obj, path, verbose = TRUE)
```

## Arguments

- obj:

  An `alccdf_linked_programs_enrolled` object

- path:

  Output file path

- verbose:

  Logical; print progress messages? Default TRUE.

## Value

Invisible file path
