# ALccdfDB: Alabama CCDF Administrative Data Cleaning and Management

A modular toolkit for cleaning, validating, linking, and managing
Alabama CCDF (Child Care and Development Fund) administrative records
from the Alabama Department of Human Resources (DHR). Features
data-driven column mapping for heterogeneous Excel report formats,
purpose-built cross-module linkage with diagnostics, Melissa.com
geocoding integration (lat/lng, census tract, FIPS), proper factor
levels for categorical variables, DuckDB-based persistent storage, and
multiple export formats (CSV, Excel, Stata, Parquet, RDS).
