# Convert time string to POSIXct datetime

Convert time string to POSIXct datetime

## Usage

``` r
.time_to_posixct(time_str, base_date_str)
```

## Arguments

- time_str:

  Character vector of HH:MM:SS formatted times

- base_date_str:

  Date string (YYYY-MM-DD) to use as the base

## Value

POSIXct vector
