# Export program data to Parquet

Export program data to Parquet

## Usage

``` r
program_export_parquet(obj, path, verbose = TRUE)
```

## Arguments

- obj:

  A program-module S3 object (any stage) or a data frame

- path:

  Output file path

- verbose:

  Logical; print progress messages? Default TRUE.

## Value

Invisible file path
