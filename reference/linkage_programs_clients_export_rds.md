# Export linked programs-clients data to RDS

Export linked programs-clients data to RDS

## Usage

``` r
linkage_programs_clients_export_rds(obj, path, verbose = TRUE)
```

## Arguments

- obj:

  An `alccdf_linked_programs_clients` object

- path:

  Output file path

- verbose:

  Logical; print progress messages? Default TRUE.

## Value

Invisible file path
