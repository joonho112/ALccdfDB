# Export program data to Stata (.dta)

Export program data to Stata (.dta)

## Usage

``` r
program_export_stata(obj, path, verbose = TRUE)
```

## Arguments

- obj:

  A program-module S3 object (any stage) or a data frame

- path:

  Output file path

- verbose:

  Logical; print progress messages? Default TRUE.

## Value

Invisible file path
